wavves

durable multi-agent routing

Moderator-led agent work needs durable state. wavves records alignment packets, model recommendations, check records and handoff files so long-running sessions can move through files instead of chat memory.

How Work Moves

Moderator

Creates the lane, records model choices and keeps the operator-facing thread small.

Lane Orchestrator

Runs bounded waves, writes findings and returns decisions to the moderator.

Gate

Captures runnable evidence under `gate-captures/` before any result is accepted.
