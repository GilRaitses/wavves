## Problem

Waves that need server-only secrets (e.g. klosr `GOOGLE_MAPS_API_KEY` for Google
densify, Twilio SID / client secret) previously required ad-hoc `setkey.sh` +
osascript. Cursor's integrated shell has no TTY for hidden `read -s`, so paste
must happen in Terminal.app. That flow must be a first-class invocable.
