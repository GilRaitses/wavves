## TLDR

- Template should look like published high-quality examples
- Sticky notes are mandatory
- Do not hardcode API keys in HTTP nodes
- Do not repost/steal other people's workflows (ban)

## Success traits

- High quality, relevant to common needs
- Real practical use cases
- Broad appeal with enough specificity to be actionable
- Original; fill library gaps

## Description page

- Clear English, Markdown, no HTML tags
- Title: Action verb + thing + to/on/in/from + where (sentence case)
- No spammy emoji titles
- ~200 words
- Sections: Who's it for; How it works; How to set up; Requirements; How to customize
- Community nodes: self-hosted disclaimer + workflow image at top

## Workflow

- Rename all nodes
- Sticky: yellow with full description; white/neutral for steps; embed media OK
- Loom encouraged
- Credentials not inline; strip personal IDs; Set Fields for config
- Assume new n8n user; plug-and-play where possible

## Payment

- ≥3 high-quality templates before paid eligibility
- Paid: creator delivers and keeps accessible; failure may ban
