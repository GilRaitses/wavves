# Proposed project skills

Evidence-backed skill drafts from lane orchestrators land here as
`<YYYYMMDD>_<skill-slug>.md`. A proposal is a request, not an install. The
moderator (O0) reviews the evidence and asks the operator before copying
anything to `../accepted/`, a repo rule, a Cursor IDE skill, or a plugin
update. Lane orchestrators and wave subagents never install or enable a
skill directly from this directory.
